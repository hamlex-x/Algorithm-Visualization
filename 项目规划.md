# 最终成果
用户打开后是主界面，主界面应有两个主要按钮；开始可视化 与 教程按钮
教程界面以后再说
开始可视化界面点进去后应该按照数据结构展示对应算法，需要包含三个按钮，退回上一步，下一步，全部执行，以及一个步骤说明区域
点击对应算法/数据结构后，进入新界面，可以自定义创建对应的数据结构
然后点击开始算法，开始演示
总之，用户可以自己创建图、数组、链表等数据结构，并观察算法每一步如何改变数据状态。
# 核心功能
1. 通过数据创建数据结构
2. 通过可视化方式创建数据结构
3. 算法可视化
4. 单步执行 / 自动播放
5. 编辑当前数据结构
6. 显示当前步骤说明

# 预期项目结构
Algorithm-Visualization/
  │
  ├── index.html              # 唯一页面（空壳，目前为空）
  ├── style.css               # 全局样式（待建）
  ├── tsconfig.json            # TypeScript 配置
  ├── CLAUDE.md                # 项目指南
  ├── 项目规划.md              # 产品规划
  │
  ├── shared/                  #   通用的文件夹
  │   ├── types.ts             #   所有 interface（GraphNode, BfsStep…）
  │   ├── datastruct.ts        #   Queue<T>, Stack<T>, PriorityQueue<T>
  │   └── svg-utils.ts         #   createSVG, createMarker, 布局函数
  │
  ├── builders/                #   构建数据结构（将用户输入变为数据对象）
  │   ├── graph.ts             #   图（邻接矩阵 + 可视化编辑）
  │   ├── array.ts             #   数组输入
  │   ├── linked-list.ts       #   链表构建
  │   ├── stack.ts             #   栈构建
  │   ├── tree.ts              #   树构建
  │   └── heap.ts              #   堆构建
  │
  ├── algos/                   #   纯算法逻辑（将数据转为step，不修改DOM树）
  │   ├── graph.ts             #   BFS, DFS, Dijkstra, A*, Kruskal, Prim
  │   ├── sort.ts              #   冒泡/选择/插入/快排/归并
  │   ├── linked-list.ts       #   插入/删除/反转/快慢指针/判环
  │   ├── stack.ts             #   括号匹配/表达式求值/递归栈
  │   ├── tree.ts              #   前中后序/层序/BST
  │   └── heap.ts              #   堆插入/删除/堆排序/优先队列
  │
  ├── renderers/               #   渲染动画（通过Step，操作SVG/DOM）
  │   ├── graph.ts             #   图（节点变色、队列进出动画）
  │   ├── array.ts             #   数组（元素变色/交换动画）
  │   ├── linked-list.ts       #   链表指针动画
  │   ├── stack.ts             #   栈压入弹出动画
  │   ├── tree.ts              #   树节点遍历动画
  │   └── heap.ts              #   堆调整动画
  │
  └── pages/                   #   组装 builder + algo + renderer 生成完整页面
      ├── home.ts              #   首页（"开始可视化" + "教程"）
      ├── list.ts              #   数据结构/算法列表
      ├── graph.ts             #   图构建页 + 图算法演示页
      ├── array.ts             #   排序演示页
      ├── linked-list.ts       #   链表演示页
      ├── stack.ts             #   栈演示页
      ├── tree.ts              #   树演示页
      └── heap.ts              #   堆演示页
